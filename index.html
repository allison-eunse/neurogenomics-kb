    üöÄ Quick Start Guide¶

    Baselines¶

    • Z-score ‚Üí residualize ‚Üí per-modality 512-D projection ‚Üí CCA + permutations ‚Üí LR/GBDT (Gene, Brain, Fusion) ‚Üí DeLong/bootstrap

    Confounds¶

    • Age, sex, site/scanner, motion (FD), SES, genetic PCs

    Roadmap¶

    • Late fusion first ‚Üí two-tower contrastive ‚Üí EI stacking ‚Üí hub tokens/TAPE if needed

    Quick Test¶

    • Run the CCA + permutation template first; it surfaces cross-modal structure before heavier prediction/fusion experiments.

    Why CCA + Permutation?

    CCA alone will always return non-zero canonical correlations, even on shuffled data. The permutation loop builds a null distribution (re-fitting CCA after shuffling one modality within the train fold) so we can report p-values and avoid over-interpreting noise‚Äîcritical when cohorts share confounds like site or ancestry.
